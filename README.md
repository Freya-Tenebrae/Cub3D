## Using Cub3D

### Cloning

```shell
git clone --recurse-submodules https://github.com/Freya-Tenebrae/Cub3D.git
```

### Compilation

for classic Cub3D :
```shell
make
```
for advenced Cub3D :
```shell
make bonus
```
